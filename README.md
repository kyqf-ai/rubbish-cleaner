# 霜刃 FrostBlade v1.0 Beta

![Language](https://img.shields.io/badge/Language-PowerShell-blue?logo=powershell)
![License](https://img.shields.io/badge/License-MIT-green)
![Platform](https://img.shields.io/badge/Platform-Windows%207%2B-blue)

一款**纯绿化、零依赖**的轻量级 Windows 磁盘清理工具，基于 PowerShell 2.0+，支持 Windows 7 及更新版本。

## 📋 核心特性

### ✨ 零依赖设计
- 彻底剥离 JSON，清理规则通过原生 Hashtable 内嵌内存
- **不产生任何配置文件残留**，单文件即可运行
- 支持 PowerShell 2.0+，Win7+ 系统

### 🚀 性能优化
- 修复 Runspace 重复执行时的底层内存泄漏，引入严格的 `Dispose()` 回收
- 废弃数组 `+=` 操作，全链路应用 `System.Collections.Generic.List[string]`
- **高速递归目录扫描**，C# Win32 API 加速

### 🛡️ 智能降级策略
- **双轨降级**：卷影清理自动侦测 CIM/WMI 接口，适配 Win10/11 的 PowerShell 7 环境
- **三轨降级**：驱动器列表获取支持 CIM → WMI → .NET DriveInfo 自动切换

### 📊 清理项覆盖面广
- **系统清理**：临时文件、缓存、日志、驱动残留等
- **浏览器**：Chrome、Edge、Firefox、Opera 等 13+ 种浏览器
- **社交通讯**：微信、QQ、钉钉、企业微信、飞书、Teams、Slack 等
- **媒体娱乐**：网易云音乐、QQ音乐、爱奇艺、B站、Steam 等
- **开发工具**：VSCode、WPS、Adobe 全家桶等

### 🎯 清理层级

| 梯队 | 清理项 | 风险 | 默认勾选 |
|------|--------|------|---------|
| **第一梯队** | 系统临时、更新缓存、浏览器缓存、回收站等 | 零风险 | ✅ |
| **第二梯队** | 日志、诊断数据、搜索索引、ASP.NET 缓存等 | 低风险 | ❌ |
| **第三梯队** | 系统还原点、事件日志、微信媒体、Windows.old 等 | **高危** | ❌ |

### 🔒 高危项保护
- 高危清理项**红色标记**，特殊提示
- 部分项启用**清理前预览确认**，逐项勾选删除
- 支持**自动创建系统还原点**（可选）
- 支持**清理前自动关闭占用进程**（可选，需二次确认）

### 📝 日志追踪
- 实时日志缓冲，异常自动拦截记录
- **清理前预览**：显示即将删除的文件路径、大小、修改时间
- 支持日志导出，完整操作追踪

### 🔧 清理兜底方案
- **SafeClean 逐项独立处理**：避免单个被占用文件导致整体中断
- **Robocopy 镜像清空法**：支持超长路径、跳过软链接
- **VSS 卷影兜底**：CIM/WMI 失败自动转用 `vssadmin` 命令行
- **权限兜底**：`Windows.old` / `$WINDOWS.~BT` 自动 `takeown + icacls`

## 📦 安装与使用

### 下载
```bash
git clone https://github.com/kyqf-ai/rubbish-cleaner.git
cd rubbish-cleaner
```

### 方式一：直接执行 Batch 启动器
双击 **霜刃启动.bat** 文件（推荐）
- 自动申请管理员权限
- 隐藏窗口后台运行
- 启动 PowerShell 执行主脚本

### 方式二：PowerShell 直接运行
```powershell
# 以管理员身份打开 PowerShell，然后执行：
Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process -Force
& "C:\path\to\FrostBlade.ps1"
```

### 方式三：静默模式（自动化）
```powershell
# 创建系统还原点 + 自动关闭占用进程
& "C:\path\to\FrostBlade.ps1" -Silent -CreateRestorePoint -ClosePrograms

# 仅清理，不创建还原点
& "C:\path\to\FrostBlade.ps1" -Silent
```

## ⚠️ 免责声明

本工具仅面向**个人电脑用户**的轻量级磁盘清理场景。

**不建议在以下场景使用：**
- ❌ 企业/生产环境
- ❌ 关键系统服务器
- ❌ 无人值守自动化部署（高危项）
- ❌ 未经充分测试的自定义规则

**清理前建议：**
1. ✅ 备份重要数据
2. ✅ 了解各清理项含义（点击项目名称查看详情）
3. ✅ 勾选前仔细阅读高危项提示
4. ✅ 开启"清理前自动创建系统还原点"选项（推荐）
5. ✅ 在非关键时段运行

**风险评估：**
| 清理项 | 可逆性 | 数据损失风险 |
|--------|--------|------------|
| 系统临时文件 | ✅ 完全可逆 | 无 |
| 浏览器缓存 | ✅ 完全可逆 | 无 |
| 微信媒体文件 | ❌ **不可逆** | **用户可见聊天内容** |
| Windows.old | ❌ **不可逆** | **系统回滚无法进行** |
| 事件日志 | ❌ **不可逆** | **历史记录永久丢失** |

## 📋 清理项详细说明

### 第一梯队：安全必清（默认全选）
- **系统临时文件** - 自动重建，官方推荐清理
- **浏览器缓存** - Chrome、Edge、Firefox 等 13+ 种浏览器
- **缩略图与图标缓存** - 零风险
- **显卡着色器缓存** - NVIDIA、AMD、Intel 三家显卡
- **软件运行时缓存** - 微信、QQ、钉钉、飞书等
- **回收站** - 全用户

### 第二梯队：安全可选（默认不选）
- **Prefetch 预读取文件** - 系统会自动重建
- **Windows 诊断日志** - 故障排查可能需要保留
- **Windows 搜索索引** - 删除后会重新索引
- **ASP.NET 临时编译文件** - 仅开发者需要
- **系统更新隐藏缓存** - 占用空间较小

### 第三梯队：高危项（谨慎操作）
- **CompactOS 系统压缩** - 可逆，但机械硬盘慎用
- **休眠文件 hiberfil.sys** - 清除后失去快速启动/休眠功能
- **Windows 事件日志** - 历史记录永久丢失 ⚠️
- **已卸载软件残留目录** - 启发式识别，有误判风险 ⚠️
- **QQ 聊天图片/自定义头像** - **用户可见内容，非纯缓存** ⚠️
- **微信媒体文件** - **聊天中的图片/视频，深度排查** ⚠️
- **系统还原点与卷影复制** - 影响系统恢复能力 ⚠️

## 🔍 工作原理

### 1️⃣ 扫描阶段
```
启动后台 Runspace
    ↓
遍历勾选的清理项
    ↓
动态发现微信/QQ/钉钉缓存路径（三轨：注册表+配置文件+全盘物理探测）
    ↓
计算各项占用空间
    ↓
显示扫描结果
```

### 2️⃣ 预览阶段
```
点击"预览" → 弹出详细清单
    ↓
显示残留目录大小/修改时间
    ↓
显示注册表失效卸载项
    ↓
用户逐项勾选确认
```

### 3️⃣ 清理阶段
```
检查是否有未扫描的勾选项
    ↓
创建系统还原点（可选）
    ↓
关闭占用进程（可选）
    ↓
逐项清理
    ├─ 普通文件删除
    ├─ 长路径文件 → Robocopy 兜底
    ├─ 注册表项删除
    ├─ 卷影清理（CIM/WMI/vssadmin 自动降级）
    └─ 回收站清空
    ↓
清理完成 + 日志导出
```

## 🛠️ 高级参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `-Silent` | Switch | 静默模式（无 GUI）|
| `-CreateRestorePoint` | Switch | 清理前创建系统还原点（需 `-Silent`）|
| `-ClosePrograms` | Switch | 清理前自动关闭占用进程（需 `-Silent`）|

**示例：**
```powershell
# 完整自动化清理
& "C:\Tools\FrostBlade.ps1" -Silent -CreateRestorePoint -ClosePrograms
```
